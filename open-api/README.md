## open-api
用于管理API的开放平台

本项目在 showDoc (https://www.showdoc.cc/) 的基础上的二次开发，前端做了些修改，后端用java进行了重写。

```
open-api
│
├─open-api 后端接口
│  
│─open-api-web 前端项目

```
